Adaptive colors In case you want your colors to be more accessible, you should consider using the hue() helper to create adaptive colors. Adaptive colors are colors that change based on the current user scheme and contrast mode. Press “s” to open Settings panel and try to change the scheme to see the difference. Predefined colors There some predefined adaptive colors like and . Themes Use definition to create a set of colors for various scenarios, schemes and modes. All generated colors are adaptive and inherit base hue and saturation. You can also apply modifiers to your theme to make it more suitable for the specific case: You can create named themes and mix them using nu-props definition. To learn more see our Themes Section or the full references of nu-theme definition and theme style attribute. Block styling Almost everything you do with CSS can be accomplished using Numl.
